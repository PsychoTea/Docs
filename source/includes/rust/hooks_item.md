# Item Hooks

## CanAcceptItem

``` csharp
void CanAcceptItem(ItemContainer container, Item item)
{
    Puts("CanAcceptItem works!");
}
```

 * Called when attempting to put an item in a container
 * Returning true overrides default behavior
